# Solution0036 Valid Sudoku

考驗寫程式的功力  
檢查 row, column 很簡單  
檢查 9宮格的數字就要好好想想了

我的做法是將 row, column, 9宮格拆成三個迴圈  
9宮格用陣列 [-1, -1], [-1, 0], [-1, 1], [0, -1], [0, 0], [0, 1], [1, -1], [1, 0], [1, 1]

別人的做法有將三個迴圈合併，看起來就很難懂
- 外圈 i，0~9
  - 外圈 j，0~9
    - 檢查 row，matrix[i][j]
    - 檢查 column，matrix[j][i]
    - 檢查 9宮格，matrix[((i/3)*3 + j/3)][((i%3)*3) + j%3] ==> 超難想...

### 複雜度

#### Time: O(1)

#### Space: O(1)
