# Solution0503

## Stack

題目的範例給不夠完整..  
我以為輸入 [1, 2, 3, 4, 3, 5]  
應該是回傳 [2, 3, 4, 5, 4, -1], 重覆的要以第一個為準    
結果答案是 [2, 3, 4, 5, 5, -1]

跟 [Solution0739](Solution0739.md) 類似，較複雜一些

雖然是環狀的陣列，不過只要繞完兩圈就可以了，可以將 nums 長度擴充兩倍，然後將數值複製兩次到新陣列，最後取得的陣列就只拿前半段

或是將迴圈設為 2*nums，然後存取陣列的地方都加上 i % nums.Length

### 複雜度

#### Time: O(n)

#### Memory: O(n)
