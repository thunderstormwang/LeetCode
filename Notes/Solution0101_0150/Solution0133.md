# Solution0133 Clone Graph

## Depth First Seach

走法跟走 tree 很像，但需要記錄已走過哪些節點，不然就進入無窮迴圈了

### 複雜度

#### Time: O(n)
- n 個節點都要走過

#### Memory: O(n)
- 最多遞迴到 n 層

這題用 BFS 去想，卡了許久

走法跟走 tree 很像，但需要記錄已走過哪些節點，不然就進入無窮迴圈了


### 複雜度

#### Time: O(n)
- n 個節點都要走過
